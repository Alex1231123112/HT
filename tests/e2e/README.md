# E2E Smoke-чеклист

- Работает сценарий регистрации бота через `/start`.
- Команда `/help` возвращает справку.
- Контент показывается в соответствии с сегментом пользователя.
- Работают вход в админ-панель и загрузка дашборда.
- Создается акция и проверяется выдача в боте.
- Создается рассылка и отправляется всем пользователям.
- Работают отмена рассылки и повторная отправка.
