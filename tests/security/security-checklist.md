# Чек-лист безопасности

- Проверить защиту от SQL injection через ORM и параметризацию запросов.
- Проверить валидацию всех загрузок по MIME-типу и размеру.
- Убедиться, что `X-CSRF-Token` обязателен для mutating-endpoint'ов админки.
- Убедиться, что попытки входа ограничены (5 за 10 минут на IP).
- Проверить срок жизни JWT и отклонение невалидных токенов.
- Проверить, что `logout` отзывает токен.
- Проверить ролевой доступ к endpoint'ам backup/export.
