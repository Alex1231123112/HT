# Руководство администратора

## Вход
- Откройте frontend админ-панели и войдите с настроенными учетными данными администратора.

## Разделы панели
- Dashboard
- Users
- Promotions
- News
- Deliveries
- Mailings
- Analytics
- Settings
- Logs

## Управление пользователями
- Фильтруйте и ищите пользователей.
- Деактивируйте невалидных пользователей.
- Экспортируйте пользователей в CSV через API.

## Управление контентом
- Создавайте акции, новинки и приходы.
- Указывайте тип аудитории (`horeca`, `retail`, `all`).

## Рассылки
- Создавайте черновики и отложенные рассылки.
- Делайте предпросмотр перед отправкой.
- При необходимости отменяйте или повторяйте отправку.
- Проверяйте статистику после отправки.
